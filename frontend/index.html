What we look for

Security checks that teach while they scan

Secret Detection

Find hardcoded API keys, database credentials, SSH keys, private certificates, and access tokens.

Why it matters

Exposed keys can lead to data breaches, account takeover, and unexpected cloud bills.

Risky Files

Detect .env files, service account JSON, PEM files, local databases, and private config files.

Why it matters

These files often contain environment-specific secrets or infrastructure details.

Vulnerable APIs

Spot plain HTTP calls, secrets in query strings, wildcard CORS, and exposed debug routes.

Why it matters

Unprotected endpoints can be abused to read, change, or delete application data.
