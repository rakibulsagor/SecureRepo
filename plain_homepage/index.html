What it does

SecureRepo functions

Each feature is separated by responsibility so the Python backend stays easy to extend.

Scanner

Secret Detection

Finds leaked API keys, cloud credentials, SSH keys, tokens, and private certificates before they reach production.

Scanner

Risky File Detection

Flags committed .env files, service account JSON files, local databases, private keys, and authentication config files.

Scanner

Vulnerable API Checks

Looks for insecure API usage such as secrets in query strings, plain HTTP requests, exposed debug routes, and wildcard CORS.

Scanner

Software Version Checks

Reads runtime and package files to detect outdated frameworks, old runtimes, and risky dependency versions.

Scanner

Configuration Review

Checks for weak security settings including permissive CORS, debug mode, and Docker containers that run as root.

Scanner

Beginner Mistake Scanner

Catches common student mistakes such as hardcoded local database paths, conflict markers, and unsafe defaults.

Report

Security Score

Calculates a readable 0 to 100 score and assigns a risk level based on the severity of detected findings.

Report

Student-Friendly Reports

Groups findings by severity and explains what happened, why it matters, and how to fix it in simple language.

AI

Gemini Explanations

Uses Gemini only to explain scanner results and suggest fixes. Detection remains rule-based and reproducible.

Account

Scan History

Stores previous scans for each signed-in user so students can track progress and revisit reports.

How it works

Simple scan workflow

Users submit a GitHub URL or local demo folder. Python scanners inspect files, collect issues, score risk, save the report, and return a clean dashboard response.

  1. 1Submit repository URL
  2. 2Clone or resolve local folder
  3. 3Run separated scanner modules
  4. 4Score and group findings
  5. 5Show fixes and scan history

Technology

Built with HTML, CSS, JavaScript, and Python

This static homepage uses plain web files. The product backend uses Python FastAPI with separate scanner modules for secrets, risky files, vulnerable APIs, software versions, configs, and beginner mistakes.
